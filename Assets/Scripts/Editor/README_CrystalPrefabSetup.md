# Настройка префаба Crystal1

## Автоматическая настройка

1. Откройте Unity Editor
2. В меню выберите **Tools → Setup Crystal Prefab**
3. Скрипт автоматически добавит все необходимые UI компоненты к префабу Crystal1

## Что будет добавлено:

- Компонент **Crystal** на корневой объект
- **Canvas** (World Space) для health bar
- **Outline** (Image) - черная обводка
- **Background** (Image) - белый фон
- **Fill** (Image) - зеленый fill для HP
- **HealthText** (Text) - текст с HP
- Компонент **CrystalHealthBar**
- Все ссылки будут автоматически назначены в компоненте Crystal

После запуска скрипта префаб будет полностью настроен и готов к использованию!
